## Valentine

**About:**
- Playful and silly application to make your loved one smile
- It is an questionnaire asking them to be your valentine except that they don't have a choice.

**How to install**
- Clone the Repo
- Create your local branch
- The Repo is an open source so feel free to drop your creative PRs and if they are a great addition I will merge them, or you can just clone an create your version and have fun!!
